## QUICK START (5 минут)

### Установка

1. **Скачайте приложение** на Google Play (когда выйдет) или установите APK:
```bash
adb install reflexio-24-7-v1.0.apk
```

2. **Первый запуск:**
   - Приложение попросит доступ к микрофону ➜ **Разрешить**
   - Приложение попросит доступ к уведомлениям ➜ **Разрешить**
   - Введите имя (опционально, для более точной диаризации)

3. **Вы готовы!** Запись начнется автоматически в фоне.

---

### Что происходит в фоне?

```
✅ Приложение слушает (VAD включен)
✅ Когда обнаружит речь — начинает записывать 3-сек сегмент
✅ Отправляет на сервер для транскрипции
✅ Получает текст + эмоции + задачи
✅ Сохраняет в локальную БД
✅ Удаляет аудио (privacy-first)
✅ Показывает в Diary (диарий)

Всё это без нажатия кнопок. Zero-friction.
```

### 2. 📅 Digest (Итог дня)

**Где:** вкладка "Итог дня"

**Что видите:** Автоматический резюме дня на основе всего, что вы говорили.

**Пример:**

```
📌 Reflexio Digest — 26 февраля 2026

🎯 Ключевые решения:
  • Договорились с Маратом сократить расходы на 15%
  • Решили запустить новый проект в марте
  • Согласовали встречу с клиентом на пятницу

⚡ Эмоции (за день):
  • Уверенность (8/10) — утро было мотивирующим
  • Стресс (4/10) — были сложные обсуждения
  • Позитив (7/10) — встреча закончилась хорошо

📋 Чек-листы:
  ☐ Подготовить таблицу бюджета
  ☐ Отправить письмо клиенту
  ☑️ Провести встречу с командой

📞 С кем общались:
  • Марат (30 мин) — обсуждение бюджета
  • Наталья (15 мин) — уточнение детали проекта
  • Мама (5 мин) — личный звонок

💡 Insights:
  "Вы в лучшей форме утром (10:00-12:00) для принятия сложных решений"
```

**Как это работает:**

1. Приложение собирает все транскрипты за день
2. Отправляет Claude Haiku на обогащение (автоматически в 21:00)
3. Claude извлекает:
   - ✨ Решения (decisions)
   - 😊 Эмоции (emotions)
   - 📝 Задачи (tasks)
   - 👤 Люди (speakers)
   - 💡 Insights (patterns)
4. Показывает красивый digest в приложении

**Кнопки:**
- **Export** → Скачать как PDF для печати или пересылки
- **Share** → Отправить в Slack, Email, WhatsApp
- **Archive** → Сохранить в историю (поиск по дате)

### 3. 🔍 Search & Semantic Memory

**Где:** вкладка "Аналитика" (будет в v2)

**Что вводите:** Естественный вопрос на русском.

**Примеры вопросов:**

```
"О чём я говорил с Маратом в январе?"
→ Результат: 5 разговоров, самый релевантный выше

"Какие решения я принял про финансы?"
→ Результат: все финансовые решения отсортированы по дате

"Когда я в лучшей форме для встреч?"
→ Результат: анализ ваших эмоций и решений по часам дня

"Какие люди важны для моего бизнеса?"
→ Результат: граф отношений с анализом частоты общения
```

**Как это работает:**

1. Вопрос кодируется в embedding (вектор смысла)
2. Ищется в базе всех ваших разговоров за последние 90 дней
3. Возвращаются 10 most relevant events
4. Claude Sonnet генерирует красивый ответ с цитатами и ссылками

### 5. 🎤 Voice Enrollment (Speaker Recognition)

**Где:** вкладка "Голос"

**Зачем:**
- Приложение учится отличать ВАШ голос от других людей
- При диаризации более точная идентификация
- Будущее: personalised recommendations

**Как:**

1. Нажмите **"Записать образец"**
2. Приложение попросит произнести 3-4 фразы:
   - "Привет, я Сергей"
   - "Reflexio — это моя цифровая память"
   - "Мне нравится принимать хорошие решения"
3. Приложение создает ваш голосовой профиль (GE2E embedding)
4. Сохраняет локально (не отправляет на сервер)

**Результат:** Диаризация станет точнее на 5-10%.

## FAQ

### Q: Может ли Reflexio слышать меня без интернета?
**A:** Да! VAD работает локально. Если интернета нет:
- ✅ Запись продолжается
- ✅ VAD сегментирует речь (локально)
- ✅ Аудио сохраняется в очередь
- ❌ Транскрипция начнется, когда вернется интернет
- ✅ Все данные остаются на телефоне (zero cloud dependency)

---

### Q: Что происходит с моим аудио? Сколько его хранится?
**A:** Zero-retention policy:
```
1. Вы говорите → VAD обнаружит речь
2. Аудио отправляется на сервер (3-сек сегмент)
3. Сервер транскрибирует (Whisper, локальный)
4. Transcription сохраняется (текст + metadata)
5. ✅ AUDIO DELETED (immediately after transcription)
6. Текст хранится 90 дней (затем автоудаление)

Итого: аудио на сервере всего 30 секунд.
```

### Q: Что если я хочу удалить все данные про человека?
**A:** Используйте право забытым:
```
Settings → Privacy → "Forget person" → Enter name
→ Все разговоры с этим человеком удаляются в течение 48 часов
```

Или через приложение:
```
Settings → About → "Export my data" → JSON file
→ Скачиваете все свои данные
→ Нажимаете "Delete account"
→ Всё удаляется в течение 7 дней
```
