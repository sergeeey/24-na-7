# Architecture Decision Records (ADR)

Архитектурные решения проекта Reflexio 24/7.

## Формат ADR

Каждый ADR содержит:
- **Context**: Контекст и проблема
- **Decision**: Принятое решение
- **Consequences**: Последствия решения
