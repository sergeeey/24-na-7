# System Patterns (Memory Bank)

**Источник:** лучшие практики из Golos.

## Захват аудио

- VAD на базе WebRTC VAD (webrtcvad).
- Буфер кадров с сегментацией по тишине.
- Сохранение сегментов в WAV (относительные пути, конфигурируемая директория).

## Транскрипция

- Единый интерфейс WhisperEngine: `transcribe(path)`, `transcribe_bytes(pcm, sample_rate)`.
- Использование существующего ASR 24 na 7 (faster-whisper / провайдеры).

## Стиль кода

- Комментарии на русском.
- Относительные импорты внутри пакета где возможно.
- Без хардкода абсолютных путей.
