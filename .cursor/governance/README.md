# Governance Loop — Система самоуправления Reflexio

Governance Loop автоматически управляет профилями работы системы на основе результатов аудита и метрик.

## Профили

### initial (текущий)
- Базовый уровень функциональности
- Ручное управление
- Минимальная автоматизация

### foundational
- Rules Engine активен
- Базовая валидация
- Memory Bank обновляется

### pro
- Полная автоматизация
- SAFE+CoVe валидация
- MCP Gateway активен

### automated
- Governance Loop автоматически переключает профили
- Метрики собираются и анализируются
- Self-healing активен

### self-adaptive
- Полная автономность
- Автоматическая оптимизация
- AI Reliability ≥ 0.95

## Использование

Профиль автоматически обновляется при запуске:

```bash
python .cursor/metrics/governance_loop.py --apply results
```

Или вручную через редактирование `profile.yaml`.
